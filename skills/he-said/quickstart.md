# Quickstart

Use when someone needs their first correct he-said policy quickly.

1. Define actor/resource terms and one relation.
2. Compose one smallest viable rule with `and`/`or`/`not`.
3. Evaluate with in-memory fixtures.
4. Show one allow and one deny outcome.
